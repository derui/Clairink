* Clairink
This is a split keyboard having these features:

- ZMK firmware
- 3D printed keyboard case & Rotally encoder nob
  - You can manage behavior with ZMK
- Focus Thocky sound with the Lofree's Phantom switch
- all wireless
- able to use rechageable AAA battery x2
  - I use eneloop for this
